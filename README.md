* Emacs 配置


** 文件结构
   - init.el: 配置文件入口
   - lisp: 配置文件模块
     - evil-mode.el:            evil 插件配置
     - init-cpp-mode.el:        针对C++的优化
     - init-org.el：            针对 Org-mode 的优化
     - init-ui.el：             视觉配置
     - init-better-defaults.el：增强内置功能
     - init-keybindings.el：    按键绑定
     - init-packages.el：       插件管理
